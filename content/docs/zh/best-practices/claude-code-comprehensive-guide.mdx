## 🚀 快速上手：基础功能掌握

### 1. VSCode 插件安装与配置

首先需要在 VSCode 中安装 Claude Code 插件。

![VSCode 插件安装](/images/claude-code-comprehensive-guide/vs-code-plugin.jpg)

**安装步骤：**
1. 打开 VSCode
2. 进入扩展商店（Ctrl/Cmd + Shift + X）
3. 搜索 "Claude Code"
4. 点击安装并重启 VSCode
5. 使用 Anthropic 账号登录

**🔗 高级配置参考：** [33 个 Claude Code 设置技巧](/docs/zh/best-practices/claude-code-setup-tips)

### 2. 第一次对话：开启 AI 编程之旅

安装完成后，可以通过以下方式开始与 Claude Code 对话：
- 快捷键：`Ctrl/Cmd + Shift + P`，然后输入 "Claude"
- 侧边栏：点击 Claude Code 图标
- 右键菜单：在文件或代码中右键选择相关选项

**实战技巧：** 首次使用时，建议先用简单的代码生成任务测试，比如 "写一个 Hello World 程序"，以熟悉交互流程。

### 3. 基础对话模式

Claude Code 支持多种对话模式：
- **代码生成**：描述需求，生成相应代码
- **代码解释**：选择代码后询问功能和逻辑
- **代码优化**：请求改进现有代码性能或可读性
- **问题诊断**：帮助定位和修复 bug

将 Claude Code 融入 CI/CD 流程：

```yaml

    runs-on: ubuntu-latest

          # 集成 Claude Code API 进行代码分析
```
